### 如何使用档案馆

档案馆的内容分为三个部分：

- 文本
- 代码，借助 github
- 图，借助 draw.io

如果文本使用到了代码或者图片，通常在文本的最前面会有一个标题为 **资料** 的部分。

- 如果是类似 **{xxx}/aaa/bbb/** 这种格式的，就是 github 的项目的代码，需要去 github 上查看。
- 如果是类似 **xxx.drawio.html** 这种格式的，就是 draw.io 生成的 html，直接打开就可以预览。

参考到的外部资料通常在文本的最后面会有一个标题为 **reference（参考）** 的部分。
