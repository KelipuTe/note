发表于： 2022-04-28
字数：564 阅读：≈ 2分钟 浏览：

环境

  • Windows 11 家庭版
  • Ruby 3.1.2
  • RubyGems 3.3.7
  • Jekyll 4.2.2

jekyll-TeXt-theme

Github： https://github.com/kitian616/jekyll-TeXt-theme

官方文档： https://tianqi.name/jekyll-TeXt-theme/

第 1 步，从 Github 上把项目下载下来。

第 2 步，安装 Ruby 依赖包。
